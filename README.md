A clean, modern, and responsive portfolio website built to showcase my journey as a Computer Science student.

##  Features
* **Responsive Design:** Works perfectly on Mobile, Tablet, and Desktop.
* **Glassmorphism UI:** Modern aesthetic with subtle blurs and shadows.
* **Interactive Contact Form:** Sends messages directly to a Google Sheet.
* **Live CV Download:** Easy access to my latest resume.
* **Hover Animations:** Smooth interactive elements for a premium feel.

##  Tech Stack
* **HTML5** - Structure
* **CSS3** - Custom styling & Animations
* **JavaScript** - Form handling & interactivity
* **GitHub Pages** - Hosting
